VIA (Variability In Atmosphere) SPECTROMETER - TELEMETRY LIST
═══════════════════════════════════════════════════════════════════

TELEMETRY ITEM          TYPE        UNITS       DESCRIPTION
───────────────────────────────────────────────────────────────────
SD_CARD_STATUS          Boolean     -           SD card enabled/disabled
AUTO_MODE_STATUS        Boolean     -           Auto mode running/stopped
AUTO_INTERVAL           Integer     seconds     Auto mode interval
MEASUREMENT_COUNT       Integer     count       Total measurements taken
SYSTEM_UPTIME           Integer     seconds     Time since boot
SPECTRUM_DATA           Array       counts      2048 pixel intensities
DEVICE_ID               String      -           Spectrometer identification
LAST_MEASUREMENT_TIME   Integer     ms          Timestamp of last measurement

═══════════════════════════════════════════════════════════════════

DETAILED TELEMETRY DESCRIPTIONS
───────────────────────────────────────────────────────────────────

SD_CARD_STATUS
  Type:     Boolean (true/false)
  Value:    ENABLED or DISABLED
  Source:   Global variable 'useSD'
  Command:  status
  Update:   On sd-on/sd-off command

AUTO_MODE_STATUS
  Type:     Boolean (true/false)
  Value:    RUNNING or STOPPED
  Source:   Global variable 'autoMode'
  Command:  status
  Update:   On auto/stop command

AUTO_INTERVAL
  Type:     Unsigned Long Integer
  Units:    Seconds
  Range:    1 - 4294967 seconds (~49 days)
  Default:  100 seconds
  Source:   Global variable 'autoInterval'
  Command:  status (when auto mode active)
  Update:   On auto command with parameter

MEASUREMENT_COUNT
  Type:     Integer
  Units:    Count
  Range:    0 - 2147483647
  Source:   Global variable 'measurementCount'
  Command:  status
  Update:   Increments on each measurement
  Reset:    On system reboot

SYSTEM_UPTIME
  Type:     Unsigned Long Integer
  Units:    Seconds
  Range:    0 - 4294967 seconds (~49 days)
  Source:   millis() / 1000
  Command:  status
  Update:   Continuous
  Reset:    On system reboot
  Rollover: ~49 days

SPECTRUM_DATA
  Type:     Array of 2048 unsigned 16-bit integers
  Units:    ADC counts (arbitrary units)
  Range:    0 - 65535 per pixel
  Size:     4096 bytes (2 bytes per pixel)
  Format:   Little-endian 16-bit values
  Source:   AvaSpec measurement buffer
  Storage:  meas_XXXX.txt (hex), spectrum_XXXX.csv (decimal)
  Update:   On each measurement
  Access:   getMeasurementBuffer()

DEVICE_ID
  Type:     String/Binary
  Units:    -
  Source:   AvaSpec identification response
  Command:  identify
  Update:   On identify command
  Contains: - Serial number
            - Hardware version
            - Firmware version
            - Device type

LAST_MEASUREMENT_TIME
  Type:     Unsigned Long Integer
  Units:    Milliseconds
  Source:   Global variable 'lastMeasurementTime'
  Update:   On each measurement in auto mode
  Use:      Calculate next measurement time

═══════════════════════════════════════════════════════════════════

DATA FILES (SD CARD TELEMETRY)
───────────────────────────────────────────────────────────────────

meas_XXXX.txt
  Type:     Binary measurement dump
  Format:   Hexadecimal ASCII
  Size:     ~8-12 KB per file
  Content:  - 10-byte header
            - 4096 bytes pixel data
            - Event log messages

spectrum_XXXX.csv
  Type:     CSV spectrum data
  Format:   Pixel,Intensity (newline separated)
  Size:     ~60 KB per file
  Rows:     2049 (header + 2048 pixels)
  Content:  Pixel index (0-2047), Intensity (0-65535)

TELEMETRY MONITORING
───────────────────────────────────────────────────────────────────

Real-time Console Monitoring:
  - Connect via USB Serial (115200 baud)
  - Type 'status' for snapshot
  - Monitor measurement progress messages
  - Auto mode shows countdown

File-based Telemetry:
  - Download files using file_transfer.py
  - Parse CSV files for spectrum data
  - Extract metadata from .txt files

═══════════════════════════════════════════════════════════════════

TELEMETRY RATES
───────────────────────────────────────────────────────────────────

Status Updates:      On-demand (status command)
Spectrum Data:       Per measurement (~15 seconds)
Auto Mode Updates:   Per interval (configurable)
Console Messages:    Real-time (continuous)
